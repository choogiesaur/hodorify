# hodorify
hodor? hodor.

This is an app that translates an English sentence into a Hodorly Decree.

##example usage:

> python hodor.py

> Enter the text to be dictated by his Hodorliness:
> Hello, my name is chika chika slim SHADY!

> His Hodorly Decree:
> Hodor, ho hodo ho hodor hodor hodo HODOR!
